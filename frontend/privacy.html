Política de Privacidad

Última actualización: 2026-04-22 · Vigente desde 2026-04-22

En RX PRO ("nosotros", "la plataforma", "el servicio") respetamos tu privacidad. Esta política describe qué datos recolectamos, cómo los usamos y tus derechos sobre ellos.

1. Datos que recolectamos

1.1 Información de la cuenta

  • Email (opcional — solo si lo proporcionás para recibir la license key y soporte)
  • Código de licencia VIP (código único generado al pagar)
  • Hash del fingerprint del dispositivo (para prevenir multi-activación abusiva)

1.2 Información de trading (si conectás broker real)

  • API Keys de Binance Futures — encriptadas en reposo con AES-256-GCM
  • Historial de trades ejecutados (símbolo, lado, cantidad, PnL, timestamps)
  • Balance de la cuenta en el momento de conexión

1.3 Información técnica

  • Dirección IP (anonimizada a /24 IPv4 o /64 IPv6 después de rate-limiting)
  • User-Agent (para detectar cambio de dispositivo sospechoso)
  • Cookies esenciales: token JWT de sesión, refresh token, preferencias UI
  • Cookies de analytics (opcional, con tu consentimiento): Firebase Analytics

2. Propósito del uso

  • Autenticación y autorización: validar que tu licencia VIP es válida y no está expirada/revocada
  • Ejecución de trades: cuando conectás Binance, usamos tus API keys para ejecutar órdenes en tu nombre (nunca para retirar fondos — el permiso de withdraw debe estar deshabilitado)
  • Soporte al cliente: tu email nos permite enviarte tu código, responder tickets y notificarte de cambios críticos
  • Mejora del producto: analytics agregados para entender uso (nunca individualizado)
  • Seguridad: detección de abuso, rate-limiting, audit log de acciones admin

3. Retención de datos

  • License keys + configuración: durante la vigencia de tu suscripción + 90 días post-expiración
  • Historial de trades: 180 días rolling, después se elimina automáticamente
  • Audit logs: 90 días (requerido por auditoría interna)
  • API keys encriptadas: se eliminan permanentemente cuando desconectás el broker
  • Logs de servidor: 30 días (con IPs anonimizadas)

4. Compartición con terceros

No vendemos ni alquilamos tus datos personales. Compartimos datos mínimos con procesadores necesarios:

  • Stripe / MercadoPago / NOWPayments: procesadores de pago. Recibimos solo confirmación de pago, no los detalles de tu tarjeta
  • Binance: cuando vos mismo conectás tu cuenta. Las API keys viajan solo entre vos, nuestro backend y Binance
  • Supabase (PostgreSQL hosted): nuestro proveedor de base de datos. Los datos están encriptados en tránsito (TLS) y en reposo
  • Render (hosting backend): nuestro proveedor de compute. Procesa tus requests pero no persiste datos
  • Google Firebase: autenticación opcional Google Sign-In (si elegís esa opción) + analytics opt-in

5. Tus derechos (GDPR / LGPD / CCPA)

Tenés derecho a:

  • Acceso: obtener copia de tus datos escribiéndonos a soporte
  • Rectificación: pedir que corrijamos datos incorrectos
  • Eliminación ("derecho al olvido"): eliminar tu cuenta y datos asociados vía email
  • Portabilidad: exportar tus datos en formato JSON
  • Oposición: desactivar cookies de analytics sin afectar funcionalidad esencial
  • Revocación de consentimiento: gestionar cookies desde el banner en landing page

6. Seguridad

  • Encriptación en tránsito: TLS 1.2+ para toda comunicación
  • Encriptación en reposo: AES-256-GCM para API keys de Binance
  • Passwords nunca se almacenan en texto plano (usamos bcrypt con cost factor 12)
  • JWT tokens con secret rotado regularmente
  • Rate limiting en endpoints sensibles (auth, trading)
  • Audit log de acciones administrativas

7. Menores de edad

El servicio no está disponible para menores de 18 años. Si descubrimos que hemos recolectado datos de un menor, los eliminaremos inmediatamente.

8. Transferencias internacionales

Los datos pueden procesarse en servidores fuera de tu país (Estados Unidos, Europa). Los proveedores que usamos cumplen estándares GDPR-compatibles.

9. Cambios a esta política

Podemos actualizar esta política. La versión vigente siempre estará en rxtrading.net/privacy.html. Notificaremos cambios materiales por email a usuarios registrados con al menos 30 días de aviso.

10. Contacto

Para ejercer tus derechos, reportar incidentes o consultas generales sobre privacidad:

  • Email de contacto: privacy@rxtrading.net
  • Soporte general: support@rxtrading.net

Al usar RX PRO aceptás esta Política de Privacidad junto con nuestros Términos de Servicio y Política de Cookies.
